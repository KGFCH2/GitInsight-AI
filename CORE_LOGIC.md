# 🧠 Core Logic & Scoring Engine — GitInsight AI

GitInsight AI operates on a sophisticated heuristic engine coupled with Large Language Model (LLM) intelligence to quantify engineering impact and provide actionable growth paths.

---

## 🔢 1. Dimensional Scoring Heuristics

The "GitInsight Score" (0-100) is calculated via a multi-layered algorithm that weights industry-standard Git metrics:

| Dimension | Weight | Logic Base |
| :--- | :--- | :--- |
| **Popularity** | 25% | Sum of Stars and Forks on **Original Repositories**. |
| **Activity** | 20% | Commit frequency, push events, and streak consistency. |
| **Quality** | 20% | README structure, licensing, and metadata (Topics/Tags). |
| **Breadth** | 15% | Language diversity and tech-stack specialization. |
| **Community** | 10% | Follower velocity and network engagement. |
| **Tenure** | 10% | Account age and long-term platform consistency. |

---

## 🏆 2. Achievement Badge Logic

Achievements are triggered by reaching specific data thresholds in the GitHub ecosystem:

- **⭐ Star Collector**: Accumulated **100+ Total Stars** across all repositories.
- **🚀 Open Source Hero**: Reached a prestigious **1,000+ Total Stars**.
- **💻 Polyglot**: Demonstrated proficiency in **5+ distinct** programming languages.
- **🔥 Consistent Contributor**: Maintained activity with **3+ repositories** updated in the last 30 days.
- **👥 Community Builder**: Cultivated an audience of **50+ Followers**.
- **📁 Prolific Creator**: Successfully managed and maintained **20+ Public Repositories**.
- **💎 Elite Profile**: Achieved a total system score of **75+**.
- **🥇 Top Repo Builder**: Built at least one "flagship" repository with **50+ Stars**.
- **✨ Rising Star**: Scored **50+ points** within the very first year on GitHub.
- **🛡️ Veteran Coder**: Maintained an active presence on the platform for **5+ years**.

---

## 🤖 3. AI Intelligence Pipeline

When deep analysis is initiated, the system passes curated profile metadata to the AI provider (Gemini/Groq):

1.  **Recruiter Insight**: The AI synthesizes metrics into a narrative that highlights technical seniority and cultural fit.
2.  **Growth Strategy**: Identifies "Improvement Vectors" (e.g., lack of documentation, stale repos).
3.  **Repository Tips**: Provides specific README and structure suggestions for the user's top repository.
4.  **Ideas Engine**: Generates 5 personalized project ideas based on the user's most frequently used languages.

---

## 📊 4. Statistics Definitions

To ensure a "Recruiter-Ready" experience, we apply strict metric definitions:
- **Stars**: *"All Stars Earned by users"* — A global measure of influence.
- **Forks**: *"All Forked Repos"* — Indicates code reach and utility.
- **Repositories**: *"Only Own Repos, excluding Forked"* — Measures original creative output.
- **Languages**: *"Most Used Languages"* — Visualizes tech-stack breadth.
