# GitInsight AI — Premium GitHub Profile Analyzer

GitInsight AI is a gamified, AI-powered platform designed to help developers showcase their impact, analyze their growth, and connect with recruiters through data-driven insights.

![GitInsight AI Favicon](/public/favicon.png)

## 🚀 Core Features

- **AI-Powered Scoring**: Get a 0-100 score based on popularity, activity, quality, and breadth.
- **Recruiter View**: Exclusive AI insights on how a recruiter perceives your GitHub profile.
- **Ambassador Program**: Gamified XP system and Top 10 leaderboard for community leaders.
- **Achievement Badges**: 10 unique 3D badges to showcase your technical milestones.
- **Professional Reports**: One-click PDF export with high-contrast, professional layouts.
- **Admin Terminal**: Secure administrative dashboard for managing the community registry.

## 🛠️ Technology Stack

- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS, Framer Motion (Animations)
- **Icons**: Lucide React
- **Analytics**: AI-driven context processing via GitHub API
- **Reporting**: jsPDF Engine
